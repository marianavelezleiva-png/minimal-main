# 🌍 EcoFuture

## Descripción

EcoFuture es una aplicación web desarrollada con Flask que promueve la conciencia ambiental mediante un cuestionario interactivo sobre hábitos sostenibles.

## Características

- 🌱 Página principal informativa.
- 📖 Sección "Acerca de".
- 📝 Cuestionario ambiental.
- 📊 Cálculo de un EcoScore.
- 💚 Recomendaciones personalizadas según el resultado.
- 📱 Diseño adaptable para computadores y dispositivos móviles.

## Tecnologías utilizadas

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5

## Instalación

1. Clonar el repositorio.

```bash
git clone https://github.com/TUUSUARIO/EcoFuture.git
```

2. Instalar las dependencias.

```bash
pip install -r requirements.txt
```

3. Ejecutar la aplicación.

```bash
python application.py
```

4. Abrir el navegador en:

```
http://127.0.0.1:5000
```
